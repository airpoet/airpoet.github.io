2.2 散列

一个散列由多个域值对(field-value pair)组成，散列的键和值都可以是文字、整数、浮点数或者二 进制数据。

同一个散列里面的各个域必 须是独一无二的，但不同域的 值可以是重复的。

尽量使用散列键而不是字符串键来储存键值对数据，因为散列键管理方便、能够避免键名冲突、并且还能够节约内存。
