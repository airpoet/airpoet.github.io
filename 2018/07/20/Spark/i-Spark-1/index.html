一. Spark 简介

1.概念

​ Lightning-fast cluster computing。

​ 快如闪电的集群计算。

​ 大规模快速通用的计算引擎。

​ 速度: 比hadoop 100x,磁盘计算快10x

​ 使用: java / Scala /R /python

​ 提供80+算子(操作符)，容易构建并行应用。

​ 通用: 组合SQL ，流计算 + 复杂分析。 

DAG:     direct acycle graph,有向无环图。
